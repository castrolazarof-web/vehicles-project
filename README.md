# Dashboard de Anuncios de Venta de Coches

## Descripción del proyecto

Esta aplicación web, construida con Streamlit, permite explorar de forma interactiva un
conjunto de datos de anuncios de venta de coches en Estados Unidos (vehicles_us.csv).

## Funcionalidad

- Visualización de un histograma de la distribución del kilometraje (odómetro) de los vehículos anunciados.
- Visualización de un gráfico de dispersión que muestra la relación entre el odómetro y el precio de venta.
- Los gráficos se generan de forma interactiva mediante casillas de verificación.

## Aplicación desplegada

https://vehicles-project-qxgd.onrender.com/

pip install -r requirements.txt
streamlit run app.py
